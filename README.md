An interactive data analytics dashboard analyzing the Guam National Tennis Team's performance in Davis Cup competitions from 2018-2025. This project demonstrates end-to-end data analysis skills including data collection, processing, visualization, and insight generation.

## Project Overview

This analytics platform processes match statistics from 29 Davis Cup matches spanning 7 years, with detailed performance metrics from 9 matches in the 2025 season. The dashboard provides strategic insights for team performance optimization through statistical analysis and interactive visualizations.

##  Key Features

- **Historical Performance Analysis**: Track win rates, match outcomes, and performance trends from 2018-2025
- **Detailed Statistical Breakdown**: Deep dive into serving statistics, break point conversion, and player-specific metrics
- **Comparative Analysis**: Side-by-side comparison of Guam players vs international opponents
- **Interactive Visualizations**: 8+ dynamic charts including bar charts, pie charts, scatter plots, and trend lines
- **Player Performance Tracking**: Individual player statistics with match-by-match detail

## Analysis Highlights

### Overall Performance Metrics
- **Total Matches Analyzed**: 29 Davis Cup matches
- **Overall Win Rate**: 44.8% (13 wins, 16 losses)
- **Peak Performance**: 2021 season (100% win rate, 4-0 record)
- **Most Frequent Opponents**: Bahrain, Turkmenistan, Mongolia

### 2025 Detailed Statistics
- First serve percentage analysis
- Ace and double fault tracking
- Break point conversion rates
- Points won distribution
- Player-by-player performance comparison

## Technical Stack

- **Frontend**: React.js
- **Data Visualization**: Recharts library
- **Styling**: Tailwind CSS
- **Data Processing**: JavaScript (statistical calculations, data aggregation)
- **Data Source**: Davis Cup official match statistics

## Key Insights

1. **Serve Efficiency**: Identified that Guam players average 64.3% first serve percentage vs 68.7% for opponents
2. **Double Faults**: Highlighted area for improvement with 5.1 average double faults per match
3. **Break Point Conversion**: Discovered correlation between break point success (46.1%) and match outcomes
4. **Historical Trends**: 2021 represents peak performance year with strategic implications for training focus
5. **Player Strengths**: Junhyuk Seo demonstrates highest first serve won percentage (68.3%)
